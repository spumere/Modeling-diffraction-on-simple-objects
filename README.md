# Моделирование дифракции на простых объектах методом углового спектра

Программа моделирует распространение плоской световой волны через различные препятствия (нить, непрозрачный диск, отверстие, непрозрачный прямоугольник) с использованием **метода углового спектра** (Angular Spectrum Method, ASM) по методике из книги Дж. Гудмена «Введение в фурье‑оптику».

## Функциональность

Проект позволяет:
* создавать плоскую световую волну;
* моделировать прохождение волны через разные препятствия:
  * нить (тонкое непрозрачное препятствие);
  * непрозрачный диск;
  * отверстие в непрозрачном экране;
  * непрозрачный прямоугольник;
* рассчитывать распространение волны на заданное расстояние с помощью метода углового спектра(теоретическое описание с источниками находится в файле asm.pdf);
* визуализировать результаты: исходную волну, волну после препятствия и волну после распространения.

## Используемые функции

* `ASM(wave, N, l, wavelength, z, inverse=False)` — реализует метод углового спектра для моделирования распространения волны.
* `plane_wave(N)` — создаёт плоскую волну размером N×N.
* `thread(wave)` — моделирует прохождение волны через нить (обнуляет столбцы в центре).
* `circle(wave, l, R)` — моделирует непрозрачный диск радиуса R.
* `hole(wave, l, R)` — моделирует отверстие радиуса R в непрозрачном экране.
* `rectangle(wave, l, a, b)` — моделирует непрозрачный прямоугольник размером a×b.
